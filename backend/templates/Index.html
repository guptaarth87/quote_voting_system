API Documentation

This API allows users to manage quotes.

Endpoints

Endpoint Method Description Request Body Response
/quote/random GET Retrieve a random quote. None 
{
    "id": 1,
    "quote": "Life is beautiful!",
    "author": "John Doe",
    "upvotes": 10,
    "downvotes": 2
}
/quote/<id>/vote POST Vote for a specific quote (upvote or downvote). 
{
    "upvote": true
}
or 
{
    "downvote": true
}
                            
{
    "id": 1,
    "quote": "Life is beautiful!",
    "author": "John Doe",
    "upvotes": 11,
    "downvotes": 2
}
/quote/add POST Add a new quote. 
{
    "quote": "New quote text",
    "author": "Author Name"
}
                            
{
    "message": "Quote added successfully!",
    "quote": {
        "id": 2,
        "quote": "New quote text",
        "author": "Author Name",
        "upvotes": 0,
        "downvotes": 0
    }
}
